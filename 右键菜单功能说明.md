# 右键菜单功能说明

## 新增功能

我已经为您的文件浏览器添加了右键菜单功能，现在您可以：

### 功能描述
- **右键点击任何文件或文件夹**：会弹出上下文菜单
- **选择"在文件浏览器中打开"**：会在Windows系统文件浏览器中打开选中的文件或文件夹

### 使用方法
1. 在文件树中找到您想要在系统文件浏览器中打开的文件或文件夹
2. 右键点击该项目
3. 在弹出的菜单中选择"📂 在文件浏览器中打开"
4. 系统文件浏览器会自动打开并定位到该文件或文件夹

### 技术实现
- **前端**：在FileTree组件中添加了右键事件处理和上下文菜单
- **后端**：在Electron主进程中添加了`open-in-explorer` IPC处理器
- **系统调用**：使用Electron的`shell.showItemInFolder()`方法在系统文件浏览器中显示文件

### 文件修改清单
1. `src/services/fileSystem.ts` - 添加了`openInExplorer`方法
2. `public/electron.js` - 添加了`open-in-explorer` IPC处理器
3. `src/components/FileTree.tsx` - 添加了右键菜单功能
4. `src/components/FileTree.css` - 添加了右键菜单样式

### 注意事项
- 该功能专为Windows系统优化
- 右键菜单会在点击其他地方时自动关闭
- 菜单样式与应用程序的整体设计保持一致

现在您可以更方便地在系统文件浏览器中打开文件和文件夹了！
