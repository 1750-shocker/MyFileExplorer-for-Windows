# 文件屏蔽功能使用说明

## 功能概述

我已经为您的文件浏览器添加了两个新的屏蔽功能：

### 1. 屏蔽该类型文件
- **功能**: 隐藏所有具有相同文件扩展名的文件
- **使用方法**: 右键点击任意文件，选择"🚫 屏蔽该类型文件 (.扩展名)"
- **特殊支持**: 对于类似 `.001`、`.002` 等数字后缀的分割文件，右键屏蔽任意一个数字后缀文件，系统会自动识别并屏蔽**所有数字后缀**文件（显示为“所有数字后缀如 .001等”）。
- **示例**: 右键点击`.txt`文件，选择屏蔽后，所有`.txt`文件将不再显示；右键点击 `.001` 分卷文件，会提示屏蔽所有数字后缀。

### 2. 屏蔽该文件/文件夹
- **功能**: 隐藏特定的文件或文件夹
- **使用方法**: 右键点击任意文件或文件夹，选择"🚫 屏蔽该文件"或"🚫 屏蔽该文件夹"
- **示例**: 右键点击某个文件夹，选择屏蔽后，该文件夹将不再显示

## 屏蔽管理

### 打开屏蔽管理器
- 在顶部工具栏中点击红色的"🚫"按钮
- 或者使用快捷方式打开屏蔽管理界面

### 管理屏蔽规则
屏蔽管理器分为两个部分：

#### 已屏蔽的文件扩展名
- 显示所有被屏蔽的文件扩展名
- 点击🗑️按钮可以取消屏蔽该扩展名

#### 已屏蔽的文件/文件夹
- 显示所有被屏蔽的具体文件和文件夹路径
- 点击🗑️按钮可以取消屏蔽该项目

## 技术实现

### 数据存储
- 屏蔽规则保存在用户主目录下的`.myfileexplorer-block-rules.json`文件中
- 包含两个数组：`blockedExtensions`（屏蔽的扩展名）和`blockedPaths`（屏蔽的路径）

### 工作原理
1. **前端**: React组件处理右键菜单交互和屏蔽管理界面
2. **后端**: Electron主进程处理文件系统操作和屏蔽规则管理
3. **过滤**: 在读取目录时自动过滤掉被屏蔽的项目

## 使用步骤

1. **屏蔽文件类型**:
   - 右键点击任意文件
   - 选择"🚫 屏蔽该类型文件"
   - 该扩展名的所有文件立即隐藏

2. **屏蔽特定文件/文件夹**:
   - 右键点击要屏蔽的项目
   - 选择"🚫 屏蔽该文件"或"🚫 屏蔽该文件夹"
   - 该项目立即隐藏

3. **管理屏蔽规则**:
   - 点击工具栏的"🚫"按钮
   - 在弹出的管理器中查看所有屏蔽规则
   - 点击🗑️按钮取消不需要的屏蔽

4. **恢复显示**:
   - 在屏蔽管理器中找到要恢复的项目
   - 点击对应的🗑️按钮
   - 刷新目录或重新浏览即可看到恢复的文件

## 界面优化

### 右键菜单智能定位
- 菜单会自动调整位置，确保不会超出屏幕边界
- 在小屏幕设备上自动适配菜单大小
- 支持滚动显示，防止菜单选项被截断

### 响应式设计
- 桌面端：完整菜单显示，最佳用户体验
- 移动端：菜单项目更大，便于触摸操作
- 自适应屏幕尺寸，确保在任何设备上都能正常使用

## 注意事项

- 屏蔽规则会立即生效，无需重启应用程序
- 屏蔽的文件仍然存在于文件系统中，只是在文件浏览器中不显示
- 可以随时通过屏蔽管理器恢复被屏蔽的项目
- 屏蔽规则会在应用程序重启后继续有效
- 右键菜单会智能调整位置，确保所有选项都能看到

## 快捷操作

- **快速屏蔽**: 右键菜单提供快速屏蔽选项
- **批量管理**: 屏蔽管理器支持查看和删除所有屏蔽规则
- **即时生效**: 所有屏蔽操作立即生效，无需手动刷新
